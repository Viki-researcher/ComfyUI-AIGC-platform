## 快速上手

```bash
# 方式一：Docker Compose（推荐）
cp .env.example .env && docker compose up -d
# 访问 http://localhost:3006  默认账号: admin / 123456

# 方式二：手动启动
# 后端
cd vue-fastapi-admin-main && python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt && DB_DEFAULT_CONNECTION=sqlite python run.py &
# 前端
cd art-design-pro && pnpm install && pnpm dev
```

详见 [部署指南](deployment-guide.md)。
